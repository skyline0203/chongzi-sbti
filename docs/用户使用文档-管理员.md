# 艾斯BTI - 用户使用文档（管理员）

> 管理员专属指南 — 翀子小工具后台管理

---

## 🔑 管理员权限说明

管理员可以：
- 查看数据统计（答题人数、类型分布）
- 管理广告位配置
- 后续版本将支持题库管理

---

## 🚪 管理员登录入口

### 方法一：隐藏入口（推荐）

1. 打开小程序首页
2. **快速连续点击页面底部「翀子小工具 © 2026」文字 5 次**
3. 系统检测到隐藏手势后，跳转到管理员登录页

### 方法二：直接访问

如果有管理后台页面路径，可以在微信开发者工具中直接导航到：
```
/pages/login/login?role=admin
```

---

## 🔐 管理员登录流程

### 第一步：进入登录页

通过上述方法进入管理员登录页面。

### 第二步：微信授权

1. 点击「管理员微信登录」按钮
2. 弹出微信授权弹窗，点击「允许」
3. 系统获取你的微信 openid

### 第三步：身份验证

1. 系统将 openid 发送到服务端
2. 服务端检查该 openid 是否在管理员白名单中
3. **验证通过** → 跳转管理后台
4. **验证失败** → 提示「你不是管理员，无权访问」

### 第四步：进入管理后台

验证通过后，你将看到管理后台首页，包含：
- 今日答题人数
- 累计答题人数
- 各类型占比统计
- 广告位配置入口

---

## 📊 管理后台功能

### 数据看板

| 指标 | 说明 |
|------|------|
| 今日答题 | 当天完成答题的用户数 |
| 累计答题 | 所有时间完成答题的用户数 |
| S 型占比 | 超级发疯型用户的百分比 |
| H 型占比 | 硬扛打工型用户的百分比 |
| I 型占比 | 离经叛道型用户的百分比 |
| T 型占比 | 咸鱼躺平型用户的百分比 |

### 广告位管理

1. 进入「广告配置」页面
2. 填入微信广告平台的广告单元 ID
3. 保存后立即生效

**如何获取广告单元 ID？**
1. 登录 [微信广告平台](https://mp.weixin.qq.com/)
2. 创建激励视频广告位
3. 复制广告单元 ID（格式：`adunit-xxxxxxxxxxxxxxxx`）
4. 粘贴到管理后台

---

## ⚠️ 安全注意事项

1. **不要分享管理员登录入口** — 隐藏入口的位置不要告诉普通用户
2. **定期检查白名单** — 确认管理员 openid 列表是最新的
3. **广告单元 ID 保密** — 不要公开分享，防止恶意刷量
4. **退出登录** — 在公共设备上操作后记得退出

---

## 🔧 后台维护

### 清除缓存

如果管理后台数据异常，可以：
1. 退出登录
2. 在微信中删除小程序
3. 重新搜索进入

### 更新版本

1. 在微信开发者工具中修改代码
2. 上传代码
3. 在微信公众平台提交审核
4. 审核通过后全量发布

---

## 📞 遇到问题？

- 检查网络连接
- 确认 openid 在管理员白名单中
- 查看微信开发者工具的控制台错误日志
- 联系翀子
